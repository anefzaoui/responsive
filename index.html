What is responsive?

Responsive, is making a webpage, web site compatible with different types of screens, resolutions and devices.

PC resolution is always large (for example 1024x768). While Mobile is always small (for example 320x480).
English article link

How to do it?

1) Using CSS3

Like this: 

          
        @media (max-width: 320px){
          #main{
            clear: both;
            width: 100%;
            margin-top:70px;
          }

          #sidebar{
            clear: botth;
            width: 100%;
            height: 70px;
            position: absolute;
            top: 0;
          }
          
        }
Everything here will be applied only when the width of a webpage is less or equal ( ≤ ) 320px.
